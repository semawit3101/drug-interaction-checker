About Drug Interaction Checker

The Drug Interaction Checker is a web-based system designed to help users identify potentially dangerous combinations of medications before use.

The system simulates real-world pharmacological interaction detection using computer science concepts such as graph theory and search algorithms.

How It Works

Each drug is represented as a node in a graph. Interactions between drugs are represented as edges. When a user enters two drugs, the system checks whether a direct or indirect connection exists between them.

The backend logic (to be implemented by the DSA team) uses:

  • Graph to show interaction between drugs
  • Breadth-First Search (BFS) to detect interaction paths
  • Hash Sets for fast drug validation and lookup

Purpose

This project demonstrates how computer science can be applied in healthcare to improve safety and decision-making.
